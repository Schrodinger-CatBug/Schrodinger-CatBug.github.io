# daily-meal GitHub Pages

本目录是 [daily-meal](https://github.com/Schrodinger-CatBug/daily-meal) 的 GitHub Pages 子包（`@daily-meal/site`），用于维护每日菜谱相关的静态网页。

项目中的菜谱介绍、每日采购提示、食材库存说明和健康饮食调整内容会写入 `index.html`，并通过 `Schrodinger-CatBug.github.io` 站点进行展示。每次更新菜谱内容时，都需要同步更新这个静态页面。

本目录为独立 git 仓库，以 **submodule** 形式挂载于主项目中。
